# kalshi-soccer-trading-bot
Kalshi soccer trading bot + live dashboard for in-play match-winner markets. The agent monitors selected leagues, detects score/time triggers (70'+ 2-goal lead and 80'+ 1-goal rule), executes risk-capped orders, enforces daily stop-loss, and logs every action.
